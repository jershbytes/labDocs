## How to setup

```bash
# Clone the Repo
git clone https://github.com/jershbytes/labDocs.git

# Change into the directory
cd labDocs

# Initialize the project
uv sync

# Source the venv
source .venv/bin/activate

# Start the zensical server
zensical serve
```
